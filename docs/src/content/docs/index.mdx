## Why Zero for Outlook

<CardGrid stagger>
	<Card title="Fast triage" icon="bolt">
		Vim-style movement, command palette actions, and quick message operations without leaving your keyboard.
	</Card>
	<Card title="Personalized controls" icon="setting">
		Customize shortcut bindings, toggle visual behavior, and add your own element-level shortcuts.
	</Card>
	<Card title="Optional AI workflows" icon="star">
		Summarize threads and generate shortcut labels with your own Gemini API key.
	</Card>
	<Card title="Privacy-first defaults" icon="shield-check">
		No analytics, no telemetry, and no remote calls unless you explicitly use AI features.
	</Card>
</CardGrid>
