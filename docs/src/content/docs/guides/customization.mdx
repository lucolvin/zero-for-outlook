## Keyboard mappings

You can remap these defaults:

- Undo shortcut (`z`)
- Command bar shortcut (`Cmd + K`)
- Blocked content shortcut (`Cmd + Shift + B`)

Each shortcut stores modifier flags plus a key, allowing cross-platform combinations.

## Behavior toggles

Available toggles include:

- Vim navigation
- Dark mode
- OLED mode
- Backdrop blur
- Inbox zero celebration
- Archive popup
- Options bar visibility
- AI title editing for custom shortcuts

## Visual settings

You can tune visual output for overlays and options UI:

- Accent color
- Popup opacity
- Backdrop blur behavior

## Custom shortcut workflow

1. Open the command bar.
2. Start the element picker.
3. Select a target element in Outlook.
4. Assign key combination and metadata.
5. Save and test in context.

Custom shortcuts sync through extension storage and are included in command search.

## Tips

- Keep system/browser reserved combos free to avoid conflicts.
- If a shortcut does not fire, check whether focus is in an editable field.
- Use command bar search terms that match your own shortcut names for faster recall.
