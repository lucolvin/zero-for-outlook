## Core shortcuts

- **Undo action**: `z` by default.
- **Command bar**: `Cmd + K` by default (`Ctrl + K` if remapped).
- **Blocked content toggle**: `Cmd + Shift + B` by default.

Shortcut keys are fully configurable from the options page.

## Vim-style navigation

When enabled, Vim mode allows keyboard-first movement across Outlook UI regions:

- `h` focuses the folder/sidebar region.
- `j` and `k` move through list items.
- `l` enters the message list.
- `Shift + j` / `Shift + k` extend selection in message lists.

Keys are ignored while typing in editable controls.

## Command bar

The command bar provides searchable actions for:

- Message operations (undo, summarize, snooze, unsubscribe).
- Navigation (mail, calendar, settings, and other Outlook surfaces).
- Extension toggles (dark mode, inbox zero celebration, options bar).
- User-defined custom shortcuts.

## Snooze overlay

Press `s` with a selected message to open snooze controls:

- Later today
- Tomorrow
- This weekend
- Next week
- Choose date

In Scheduled views, unsnooze actions are available.

## Custom shortcuts

Use the element picker to bind keys to buttons or UI controls directly in Outlook.

- Shortcuts are saved in extension storage.
- Custom actions appear in the command bar automatically.
- Optional AI title editing can generate cleaner names/descriptions.

## Optional AI features

AI functionality requires your own Gemini API key:

- Email summarization
- AI-generated shortcut labels

Without a key, the extension still runs fully for non-AI workflows.
