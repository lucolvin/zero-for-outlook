Zero for Outlook is a browser extension that improves Outlook Web with custom keyboard shortcuts, command-driven actions, and optional AI helpers.

The project focuses on:

- Fast triage and navigation from the keyboard.
- Configurable behavior rather than fixed workflows.
- Privacy-first defaults with optional external AI integrations.

## Compatibility

The extension targets:

- `outlook.live.com`
- `outlook.office.com`
- `outlook.office365.com`
- `outlook.cloud.microsoft`

## Project status

Zero for Outlook is actively developed and may include frequent changes across releases.
