Vim-style navigation

Toggle vim-style navigation to move efficiently between folders and messages using h, j, k, and l.

Enable vim navigation

Safe by default: keys are ignored while typing in inputs.

  • Sidebar: h (or l from the message list) focuses and “pins” the sidebar. While pinned, j / k move up and down folders in the sidebar.
  • Message list: Press l from the pinned sidebar to move into the message list; then j / k move between messages.

LLM shortcut names

LLM summary (optional)

Add a Google Gemini API key to enable an LLM command that briefly summarizes the important details from the currently open email in Outlook.

Gemini API key

Stored only in your browser’s extension storage.

  • Disclaimer: LLMs can hallucinate. Verify important details in the original email.
  • When it runs: The key is used only when you invoke Summarize current email from the command bar.
  • What is sent: Visible text of the open email is sent to Google Gemini with brief instructions.

How it works

  • Where it runs: Only on Outlook web pages (outlook.live.com, outlook.office.com, outlook.office365.com).
  • What it does: Listens for your shortcut and clicks the Undo button after you archive or move an email. With vim navigation, you can move in the list with j / k.
  • Typing safety: While composing in an input, text area, or editable field, vim navigation, Undo, Snooze, and similar shortcuts are ignored. Your command bar shortcut and custom shortcuts still work there.

Privacy

Zero for Outlook never collects, stores, or sends any of your email content or browsing data.

  • No tracking: No analytics, telemetry, or external network requests from the extension itself.
  • Local-only settings: Shortcuts and toggles live in the browser’s extension storage.
  • Minimal permissions: Runs only on Outlook web domains and simulates clicks for Undo and navigation.

Backup & restore

Export creates a JSON file of everything stored in sync (shortcuts, toggles, custom shortcuts, optional Gemini API key, and related flags). Import replaces your current sync data with the file contents after you confirm.

Export settings

Your browser downloads a dated JSON file. Keep it somewhere safe if it includes an API key. Use the same file to import on this or another profile with the extension installed.

Nothing is uploaded; the file is generated locally from extension storage.

Import settings

Pick a JSON file from this extension or a compatible backup. You will be asked to confirm; then sync storage is cleared and replaced with the file. This cannot be undone.

Prefer a file you exported from Zero for Outlook, or a plain sync object with known keys.
